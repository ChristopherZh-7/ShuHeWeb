# 舒赫安全公司网站 - 启动说明

## 项目简介
这是一个使用 React + Vite + Tailwind CSS 构建的现代化安全公司展示网站。

## 启动方法

### 方法一：使用 WebStorm IDE（推荐）
1. 在 WebStorm 中打开项目
2. WebStorm 会自动检测到 package.json
3. 在终端中运行：`npm install`
4. 安装完成后运行：`npm run dev`
5. 浏览器会自动打开 http://localhost:3000

### 方法二：解决 PowerShell 执行策略问题
1. 以管理员身份打开 PowerShell
2. 运行：`Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`
3. 输入 Y 确认
4. 然后在项目目录运行：`npm install`
5. 启动开发服务器：`npm run dev`

### 方法三：使用命令提示符（CMD）
1. 按 Win+R，输入 cmd，回车
2. 导航到项目目录：`cd "C:\Users\Christopher\WebstormProjects\ShuHeWeb"`
3. 运行：`npm install`
4. 启动：`npm run dev`

### 方法四：使用 VS Code
1. 在 VS Code 中打开项目文件夹
2. 打开终端（Ctrl + `）
3. 运行：`npm install`
4. 启动：`npm run dev`

## 项目特色功能
- 🎨 现代化蓝色科技风设计
- 📱 完全响应式设计（支持手机、平板、桌面）
- ✨ 精美的动画效果和过渡

## 项目结构
```
src/
├── components/          # 组件目录
│   ├── Navbar.jsx      # 导航栏
│   ├── Hero.jsx        # 主横幅
│   ├── Services.jsx    # 服务介绍
│   ├── About.jsx       # 关于我们
│   ├── Contact.jsx     # 联系我们
│   └── Footer.jsx      # 页脚
├── App.jsx             # 主应用组件
├── main.jsx            # 应用入口
└── index.css           # 样式文件
```

## 自定义修改
- 修改公司信息：编辑各个组件中的文本内容
- 更换配色：修改 `tailwind.config.js` 中的颜色配置
- 添加新功能：在 `src/components/` 目录下创建新组件

## 部署说明
构建生产版本：`npm run build`
预览构建结果：`npm run preview`

## 技术栈
- React 18 - 用户界面框架
- Vite - 快速构建工具
- Tailwind CSS - 实用优先的CSS框架
- Lucide React - 精美的图标库
